### 校园自助洗衣管理系统✔万字文档(LW)✔Java✔SSM✔JSP✔启动教程
## 源码 + 万字文档（LW）+  SQL脚本 + 软件环境安装包
【项目采用技术】
架构: B/S、MVC
系统环境：Windows/Mac
开发环境：IDEA、JDK1.8、Maven、MySQL、tomcat7.5/8/9
技术栈：Java、MySQL、Spring、SpringMVC、Mybatis/MybatisPlus、JSP、JavaScript、CSS、javaweb
【系统功能模块】
本系统包含管理员和用户2个角色。
①：管理员角色：
1. 洗衣机管理：管理员可以增删改查洗衣机信息。
2. 已预约洗衣机管理：管理员可以查看和取消用户预约的洗衣机。
3. 留言管理：管理员可以管理用户发布的留言，并进行回复。
4. 系统公告管理：管理员可以发布和维护系统公告。
5. 用户管理：管理员可以管理用户信息，包括查看用户余额和编辑用户资料。
②：用户角色：
1. 查看洗衣机信息：用户可以查看洗衣机的状态和相关信息。
2. 预约洗衣机：用户可以对可用的洗衣机进行预约。
3. 取消预约：用户可以取消已预约的洗衣机。
4. 发布留言：用户可以在系统中发布留言。
5. 查看公告：用户可以查看系统发布的公告。
